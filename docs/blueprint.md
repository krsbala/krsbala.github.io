# **App Name**: Profile Dark

## Core Features:

- Dynamic Content Loading: Load website content dynamically from a single `data.json` file, including text, images, and links.
- Hero Section: Implement a hero section featuring the name, tagline, and optional photo.
- Achievements Section: Showcase key achievements in a dedicated section.
- Experience Section: Present work or professional experience in a timeline or card-based format.
- Projects Section: List projects with titles, descriptions, and links to the projects.
- Contact Section: Display contact information, including email, LinkedIn, and GitHub links.
- Smooth Scrolling: Implement smooth scrolling between different sections of the single-page website.

## Style Guidelines:

- Background color: Dark grey (#282A3A) for a modern, minimal dark theme.
- Primary color: Purple (#BD93F9) for highlights and interactive elements.
- Accent color: Light green (#84a98c) for a subtle contrast.
- Font: 'Inter', a sans-serif font, for both headings and body text, providing a clean and modern look.
- Use a clean, minimal layout with clear section divisions. Utilize spacing and padding to ensure readability.
- Use simple, outlined icons from a library like FontAwesome or Feather Icons for key sections and links.
- Incorporate subtle animations, like fade-in effects, on scroll to enhance user experience without being distracting.
